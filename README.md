# C# Modbus RTU串口通信资源文件

## 描述

本资源文件提供了一个使用C#实现的Modbus RTU串口通信工具。该工具支持Modbus RTU封包格式的自动CRC计算，并包含测试功能，方便用户进行串口通信的开发和调试。

## 功能特点

- **Modbus RTU支持**：完整支持Modbus RTU协议，适用于工业自动化领域的串口通信。
- **自动CRC计算**：工具能够自动计算Modbus RTU封包的CRC校验码，确保数据传输的准确性。
- **测试工具**：内置测试功能，方便用户验证通信的正确性和稳定性。

## 使用方法

1. **下载资源文件**：从本仓库下载所需的资源文件。
2. **导入项目**：将资源文件导入到您的C#项目中。
3. **配置串口**：根据您的硬件配置，设置串口参数（如波特率、数据位、停止位等）。
4. **编译运行**：编译并运行项目，使用提供的测试工具进行通信测试。
